When to Use This

Annual reports, yearly data archival, license renewal reminders, New Year cleanup scripts.

Crontab Entry

0 0 1 1 * 0 0 1 1 * /opt/scripts/annual-archive.sh && echo 'Annual archive completed' | mail -s 'Annual Job OK' admin@company.com
0 0 1 1 * /path/to/your/script.sh

Things to Know

⚠️ Watch out

If this fails, you wait a full year for the next run. Always have monitoring and alerting on annual jobs. Test it manually before relying on the schedule.

💡 Pro tip

For important annual jobs, add a monthly check that verifies the last annual run completed successfully. Don't trust a once-a-year job to just work.
