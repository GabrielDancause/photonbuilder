When to Use This

Weekly maintenance: full backups, log archival, database vacuuming, weekly security scans.

Crontab Entry

0 0 * * 0 0 0 * * 0 /opt/backup/weekly-full-backup.sh
0 0 * * 0 /path/to/your/script.sh

Things to Know

⚠️ Watch out

Sunday can be 0 or 7 depending on the system. Both usually work in modern cron implementations, but 0 is the most portable.

💡 Pro tip

Sunday night/Monday morning is the lowest-traffic period for most business systems. Run heavy maintenance here.
