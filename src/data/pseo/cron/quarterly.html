When to Use This

Quarterly financial reports, Q1/Q2/Q3/Q4 summaries, quarterly tax estimates, compliance audits.

Crontab Entry

0 0 1 1,4,7,10 * 0 0 1 1,4,7,10 * /opt/scripts/quarterly-report.sh
0 0 1 1,4,7,10 * /path/to/your/script.sh

Things to Know

⚠️ Watch out

This fires 4 times a year. There's no 'every 3 months' syntax in cron — you list the specific months.

💡 Pro tip

Fiscal quarters vary by company. If your fiscal year starts in July, use '0 0 1 7,10,1,4 *' (though order doesn't matter in cron).
