Regex for Ipv6

A regular expression to match a Ipv6.

Regular Expression 
^(?:[A-F0-9]{1,4}:){7}[A-F0-9]{1,4}$i

Explanation

Part Description
^ Start of the string
.* Matches any character
$ End of the string

Examples
