Frequently Asked Questions

Why isn't my website loading in the preview?

Some websites use security headers (like X-Frame-Options) that prevent them from being displayed inside an iframe. If you see a blank screen or a connection error, the target website likely restricts embedding.

Are the performance metrics real?

The metrics provided in this tool are simulated to demonstrate how device-specific performance tracking works. For accurate Lighthouse scores, we recommend using Chrome DevTools or Google PageSpeed Insights directly.

Why does the screenshot fail sometimes?

Browser security policies prevent capturing screenshots of cross-origin iframes. If you are previewing a website that is not on the same domain as this tool, the screenshot engine cannot access its contents.

Does the touch simulation actually test touch events?

The touch simulation provides a visual crosshair and visual feedback, but because it relies on an iframe, it cannot intercept or alter the actual javascript event types (mouse vs touch) being fired inside the cross-origin framed document.
